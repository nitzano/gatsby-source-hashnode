## Install

```bash
npm install gatsby-source-hashnode
```

## How to use

Configure the plugin in `gatsby-config.js`:

```javascript
// gatsby-config.js
module.exports = {
  plugins: [
    {
      resolve: "gatsby-source-hashnode",
      options: {
        username: "<HASHNODE_USERNAME>",
      },
    },
  ],
};
```

## How to query

### Get user posts

```graphql
{
  allHashNodePost {
    edges {
      node {
        brief # "In this article..."
        # File node (to be used with gatsby-plugin-image)
        coverImage {
          absolutePath
          relativePath
          extension
          url # "https://cdn.hashnode.com
        }
        slug # "my-great-article"
        title # "My Great Article"
      }
    }
  }
}
```

### Get post's reading time

(With the kind help of [reading-time](https://www.npmjs.com/package/reading-time))

```graphql
{
  allHashNodePost {
    edges {
      node {
        title
        readingTime {
          minutes # 2
          text # "2 min read"
          time # 120000
          words # 100
        }
      }
    }
  }
}
```

```graphql
        numReactions # 95

        publicationDomain # "blog.mydomain.."
        tagline # "I do stuff"
        username # "username1"
      }
    }
  }
}
```
